# Dashboard: Channels & Entities

Enhanced ATC dashboard with three tabs: **Event Stream**, **Channels**, and **Entities**.

## Files

| File | Description |
|------|-------------|
| `dashboard.html` | Drop-in replacement for `/app/bootstrap/dashboard.html` |
| `refresh-channel-map.py` | Scans `.aladuo/var/channels/` + calls Feishu `im/v1/chats/{chat_id}` API to build channel ID map |
| `save-api.py` | Lightweight host-side HTTP server (port 20234) for dashboard edit/save functionality |

## Features

### Channels Tab
- **Global Config** (collapsible cards): `config/feishu.md` (editable), `memory/CLAUDE.md`, `CLAUDE.md` kernel (read-only)
- **Group Channels**: lists all known groups with status dots, names from Feishu API, and expandable `descriptor.md` (editable)
- Channel list from `channel-id-map.json` (generated by `refresh-channel-map.py`), session status from `system.status` RPC
- Chinese content correctly decoded (UTF-8 base64 via `TextDecoder`)

### Entities Tab
- Lists all entities from `memory/index.md`, grouped by section (People, Tools, Services, Projects)
- Search/filter box
- Each entity expandable with markdown rendering + Raw toggle + Edit/Save

### Edit/Save
- Uses `save-api.py` running on the host (writes to bind-mounted directories)
- Only allows writes to approved path prefixes (`aladuo/config/`, `aladuo/memory/entities/`, `.aladuo/var/channels/`)

## Setup

```bash
# 1. Generate channel ID map (requires FEISHU_APP_ID/SECRET in .env)
python3 refresh-channel-map.py

# 2. Copy dashboard into container
docker exec duoduo-openduo cp /home/duoduo/aladuo/dashboard.html /app/bootstrap/dashboard.html

# 3. (Optional) Start save-api for edit/save support
nohup python3 save-api.py > /tmp/save-api.log 2>&1 &
```

## Integration suggestion

To ship this as the default dashboard, copy `dashboard.html` to the bootstrap directory in the container image build:

```dockerfile
COPY dashboard.html /app/bootstrap/dashboard.html
```

And include `refresh-channel-map.py` in a setup script or as a daemon startup hook.
